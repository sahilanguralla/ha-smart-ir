# HACS Custom Integrations

This repository contains custom integrations for Home Assistant, compatible with HACS (Home Assistant Community Store).

## Available Integrations

### [Dyson IR Control](custom_components/dyson_ir/README.md)
A custom integration to control Dyson AM09 (and similar) fans/heaters using an IR blaster. Supports full device configuration, including IR code learning and state management.

## Installation
1.  Add this repository to HACS > Integrations > 3 dots > Custom repositories.
2.  Search for "Dyson IR" and install.
3.  Restart Home Assistant.
